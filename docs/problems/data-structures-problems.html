Programming questions

  1. Revisit your previous work on counting words in sentences. Basing yourself on the code you wrote, could you implement a function that will take as input a sentence (as a string) and split it into a list of words? This process is called tokenizing and is at the basis of natural language process, which we will be experimenting more later.

  2. Reading files can be done using lists and loops. For example, the follow snippet reads a file located in the same directory as the script:

     with open('text.txt', 'r') as myfile:
   for line in myfile:
     print(line)

    In this example, the open() function will access the file (the second parameter is the access mode, r symbolizes “reading”), and treats its content as a list of lines, hence the for loop processing each line in the file. The resulting line value is a string representing the line read by yoru script.

    Using this information and a dictionary file, write a script that reads the dictionary file and computes the distribution of words per length. The output of your script should be a list of length:count pairs.

  3. Expanding on the previous problem, can you make a lookup function that will look into the file for a given word provided by the user? You shouldn’t read the file every time and should have your program ask the user for another word once a query is processed.

  4. Can you make the last program look for similar words? For example, given a dictionary containing the word “fruity”, your program should return “fruity” amongst its results for a query looking for words similar to “fruit.”

    For simplicity, you can assume that similarity is defined as “words have a common prefix”.
