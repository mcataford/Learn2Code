Loops

True/False questions

  1. Any while loop can be rewritten as a for loop.
  2. It is impossible to create infinite loops with for loops.
  3. The range() function can be used without any parameter, in which case the default values are min=0 and max=10.
  4. Loops require the same kind of indentation as other blocks.
  5. Conditional blocks and loops cannot be nested within each other.
  6. Infinite loops are not always undesirable.

Short answer questions

  1. What is one useful case where an infinite loop is a desirable feature?
  2. Describe the different parameters of the range() function and how they affect the function’s output.
  3. How can you convert a while loop to a for loop?
  4. What do you need to be careful about in order to prevent infinite loops?

Programming questions

  1. The value Pi can be approximated using an unending summation of numbers whose result provides a good estimate of the exact value of Pi. In this problem, you will be implementing the iterative scheme called a “Gregory-Leibniz series” described here.

    Your implementation should be a function which takes for input the number of iterations your approximation should go through. It should return the final value as a float.

    Can you experimentally find how many iterations your implementation would need to reach 6 digits accuracy with respect to the exact value of Pi?

  2. Fibonacci numbers are defined by the recursive relation Fn+1 = F + Fn-1, which translates to “the next value (Fn+1) is calculated by summing up the two previous values (Fn and Fn-1).”

    Generating Fibonacci numbers can be done quite easily using loops since the process of calculating those numbers is inherently repetitive. Using loops, write a function that will compute the nth Fibonacci number knowing that the first two numbers are 0 and 1 (i.e. F0 = 0, F1 = 1). Your function’s input should be the number of number to generate, it shouldn’t return anything and instead print each value as it is generated.

    Make sure that the values printed are right using the examples cited in the page about Fibonacci numbers linked earlier.

  3. Implement a program that will prompt the user for a sentence and count how many distinct words are contained in it. You can ignore punctuation and define a word as “a contiguous series of characters without punctuation or whitespace”.

    For example, the sentence “Hello, World! Isn’t the sun bright today?” would contain 7 words.

    Note: You do not yet need to keep track of the words, but we will later revisit this when we learn how to store collections of data properly.

    You should make sure that your program uses functions to encapsulate code. As an added challenge, how would you make it so that your program asks for another sentence once it’s done?
