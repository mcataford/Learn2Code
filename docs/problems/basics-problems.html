Python basics

Variables, functions and basic programs

True/False questions

  1. The interactive Python console prints the output of an operation automatically.
  2. The ** operator implements two consecutive products such that a**b = a * b * b.
  3. Primitive data types are types of data that can be represented as one or several numbers.
  4. A string is an example of a primitive type because each character of a string can be mapped to a number via the ASCII table.
  5. Boolean values can be represented as numbers.
  6. A string is represented by a collection of characters surrounded by double quotes.
  7. + and * are two operators defined for strings; other operators are not defined and will raise an error if used with string operands.
  8. To concatenate two values, you must first convert them using string(x).
  9. You can only concatenate two values at a time.
  10. When defining variables, the data type must be mentioned as well as the name and initial value.
  11. Any statement that can be written in the interactive Python console can also be written in a script file.
  12. Outside of the interactive console, print(x) will allow you to print output to the screen.
  13. Functions are defined using the define keyword.
  14. Comments are ignored by the interpreter, as such, you can include as many as you wish.
  15. A script file can contain both space and tab indentation.
