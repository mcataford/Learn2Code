Conditional statements and branching

  1. The random module allows you to generate pseudorandom numbers at will. By using its randint(min,max) function, you can generate integers that satisfy min <= x <= max. You can find an example usage below:

       import random

   random_int = random.randint(0,100)

   #Print random integer between 0 and 100.
   print(random_int)

    Using random, design a small program that asks the user a random mathematical problem, asks for an answer and evaluates whether the user answered correctly or not. To simplify , consider a single type of mathematical operation and only make the operands vary.

    For example, your script could generate math problems of the form “a + b = ?” and use random to change a and b.

  2. Using a strategy of your choice, design an email validator function that will take a string as input and will return a boolean value describing whether the input is a valid email or not. For the purposes of this exercise, we can define a “valid email” as the following:

    1. A string containing exactly one @ symbol, separating the username and the domain of the user;
    2. The username should only contain alphanumerical characters, periods and underscores;
    3. The domain should only contain alphanumerical characters and be of the form domain.xyz where you can assume the extension to always be 3 characters;
    4. The overvall format should be username@domain.xyz.

    To accomplish this task, you may want to look at the re module or implement your own solution using string indices and conditional statements.
