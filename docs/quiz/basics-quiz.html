Variables, functions and basic programs

  1. In the problem set, you implemented a function that prints a rectangle of variable width. Add another parameter to the function such that the user can also choose the height of the rectangle.

    Hint: This can be entirely implemented using the right operator and that the \n character will insert a newline in a string, which may solve a lot of problems.

  2. The input() function allows you to query the user for information. Any parameter that is given to it will be printed as a prompt before the script stops to wait for the user to type in something. Using input(), can you modify your solution to 1. further such that it asks the user which width and height to use to generate the rectangle? You should also have your function print a sentence containing the width and height that were used.

    __Note: here is an example of how input() can be used based on the introduce_myself function we implemented before:

     #This function is taken from the Python basics lesson text, under section 4 (Functions)
 def introduce_myself(name, age, height):
   output = "Hello! My name is " + name + " and I am " + str(age) +  " years old. I am " + height + " centimetres high."

   return output

 """
 We can query the user for each of the three values we need. Each time,
 we use an appropriate prompt for the user.

 Note that the same function is used to capture any of the values! We'll discuss this again later.
 """
 user_name = input("What is your name?")
 user_age = input("How old are you?")
 user_height = input("How tall are you?")

 sentence = introduce_myself(user_name, user_age, user_height)

 print(sentence)
